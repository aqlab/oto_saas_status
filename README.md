
#### 目前的方式是这样:

  ● 网关部门定时扫描各个服务,进行监控状态检查,并通知给指定负责人.
  
  
#### 实现逻辑:
  ● 1. 要求每个线上服务暴露一个/status 地址,并返回指定json,将项目中依赖的数据库连接信息返回
  
#### 实现要求:
  ● 1. 组件式开发
  ● 2. 引入即可用,自动监控项目中数据库连接
  ● 3. 暴露一个指定的可供网关扫描的地址
  
#### 技术实现方式:
  ● 1. 引入spring-boot-autoconfigure 自动配置
  ● 2. 引入spring-boot-actuator  使用EndPoint 定制暴露给网关的扫描地址
